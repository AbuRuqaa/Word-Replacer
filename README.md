## Word-Replacer




# About
Word Replacer will let you replace words from multiple docx files from a folder and its sub folders
! [Main Window] (https://github.com/AbuRuqaa/Word-Replacer/blob/main/resources/MainWindow.png)

1-Type the word you want to replace
2-Enter the new word
3-Choose your folder path

! [MainWindow_PB] (https://github.com/AbuRuqaa/Word-Replacer/blob/main/resources/MainW_ProgressB.png)
